# Image Setup Instructions

Please save your two professional photos in the `public` folder with these names:

1. **First photo (with sunglasses, casual professional)**: Save as `photo2.jpg`
2. **Second photo (in suit, formal professional)**: Save as `photo3.jpg`

## Steps:
1. Save the first image (sunglasses photo) as `public/photo2.jpg`
2. Save the second image (suit photo) as `public/photo3.jpg`

The portfolio will automatically display all three photos in the About section with a beautiful gallery:
- Main photo: `myphoto.JPG` (already exists)
- Photo 2: `photo2.jpg` (casual professional with sunglasses)
- Photo 3: `photo3.jpg` (formal professional in suit)

Users can click on thumbnails to switch between photos!
